Key Features And Functions

  • Digital Transaction Management.
  • Send pdf documents for signature to salesperson on one button click.
  • Get salesperson's signed document on one button click.
  • Send salesperson's signed pdf documents to customer for signature on one button click.
  • Get customer's signed documents on one button click.
  • Get status of pdf documents on button click.


DocuSign Credentials

  • You need docusign developer sandbox Account.
  • Generate an integrator key login to your sandbox, click the profile icon in the top right, and select Go To Admin
  • On the lower left side of the Admin Console select API and Keys then click the Add Integrator Key button
  • In ODOO go to the DocuSign->Settings
  • Then provide Docusign credentials
    • name(provide credentials name whatever you want)
    • Email(your docusign developer sandbox account email)
    • Password(your docusign developer sandbox account password)
    • Docusign Integrator Key(your docusign developer sandbox account Integrator key)
  • Also test these credentials by click on Test credentials button.
  • Save Record


Send pdf documents for signature




  • After test and save credentials, go to sales --> quotations or sales --> customers and go to DocuSign tab.
  • Select socusign account.
  • Attach a pdf document and click send documents to salesperson button. Document will be sent to the document account that you configured earlier.
  • You can see sent document in bottom and you can also check document status.
  • Now go to docusign account and sign document.
  • Come back in ODOO and click on update status button.
  • Now you can see document's status is completed.
  • Now to send document to customer select the agreement in salesperson's signed agreement dropdown. This dropdown only contain documents that are signed by salesperson.
  • Click send documents to the customer. Document will be sent to the customer.
  • You can check if customer signed document by clicking update status button.
  • Once the customer signed document you can see all attachments(unsigned, salesperson signed, customer signed) in agreement at the bottom of the screen.


Pre requisites


  • Require a Docusign developer sandbox account.
  • Generate an Integrator Key.
  • Need to save developer sandbox account email, password and integrator key.
