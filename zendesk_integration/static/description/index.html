Odoo Zendesk Support

Odoo Zendesk Support puts all your customer support interactions in one place, so communication is seamless, personal, and efficient which means more productive agents and satisfied customers. Zendesk Support is a beautifully simple system for tracking, prioritizing, and solving customer support tickets.



Demo Video





Key Features And Functions

  • One click API integration.
  • Sync tickets from zendesk to odoo.
  • Simple module for tracking tickets.
  • Prioritizing tickets.
  • Categorize zendesk tickets in odoo.
  • Update zendesk tickets in odoo.
  • Add comment to the tickets.
  • Add private comment to the ticket.
  • Automatically sync data between zendesk and odoo.


Create Zendesk Credentials

  • Go to setting -> user and companies -> select users.
  • Open user and enter Zendesk Credentials in Zendesk Support Connection tab.
  • Enter your url, email and password and click on Test Connection/Sync Data button.
  • After syncing data now you can see tickets in zendesk main tab.


Auto Schedulers

  • Go to settings --> Technical --> Automation --> Scheduled Action.
  • Here you can see schedulers.
    • Activate Tickets Scheduler
  • *Point to Ponder: Activate Scheduler as Login User.
