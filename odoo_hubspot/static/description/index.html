Odoo Hubspot

Odoo is a fully integrated suite of business modules that encompass the traditional ERP functionality. Odoo Hubspot Connector provides the opportunity to import contacts and companies from Hubspot to ODOO.



Demo Video





Key Features And Functions

  • Import contacts from Hubspot to odoo.
  • Update contacts from Hubspot to odoo.
  • Export contacts from odoo to Hubspot
  • Update contacts from odoo to Hubspot.
  • Import companies from Hubspot to odoo.
  • Update companies from Hubspot to odoo.
  • Export companies from odoo to Hubspot.
  • Update companies from odoo to Hubspot.
  • Import leads from Hubspot to odoo.
  • Update leads from Hubspot to odoo.
  • Auto Import contacts.
  • Auto export contacts.
  • Auto import companies.
  • Auto export companies.


Hubspot Credentials

  • You must have a Hubspot account to use this connector.
  • Login to hubspot and click profile icon in top right corner. You can see integration option in theopened menu. Click it
  • Here you can see hubspot API key.
  • Now go to settings --> users & companies --> user.
  • Open user and go to Hubspot tab.
  • Paste hubspot API key in the key field and save user.


Import Companies




  • Now Go to Hubspot tab in user
  • Click Import Companies.
  • If you go to sales --> customers. You can see companies imported here.


Export Companies




  • Now Go to Hubspot tab in user.
  • Click Export Companies.
  • You can see companies expoted in your hubspot account.


Import Contacts




  • Now Go to Hubspot tab in user
  • Click Import Contacts
  • If you go to sales --> orders --> customers. You can see hubspot contacts imported here


Export Contacts




  • Now Go to Hubspot tab in user
  • Click Export Contacts
  • You can see Contacts expoted in your hubspot account


Import Leads




  • In Hubspot contacts you can mention stage of customer that can be lead, opportinities, qualified.
  • Now Go to Hubspot tab in user.
  • Click Import Contacts.
  • If you go to crm app in odoo. You can see leads and opportunities synced here.


Auto Sync




  • We provide option to automatically sync data between odoo and Hubspot.
  • Go to odoo settings --> technical --> scheduled actions --> automated actions.
  • You can see these schedulers to sync your data.
    • Auto companies export
    • Auto companies import
    • Auto contacts export
    • Auto contacts import
