ODOO Eventbrite

ODOO is a fully integrated suite of business modules that encompass the traditional ERP functionality. ODOO integration with Eventbrite gives us power to manually sync events between ODOO and Eventbrite. Use ODOO Eventbrite to sync events between ODOO and Eventbrite.



Demo Video





Key Features And Functions

  • Import events from Eventbrite to ODOO.
  • Import tickets from Eventbrite to ODOO
  • Import attendees from Eventbrite to ODOO
  • Import orders from Eventbrite to ODOO
  • Update events from Eventbrite to ODOO
  • Update tickets from Eventbrite to ODOO
  • Update attendees from Eventbrite to ODOO


Eventbrite Credentials

  • Go to settings -->users --> Administrator --> Eventbrite past auth token here that you copied earlier.
  • Click Test Connection button to check if token worked.


Pre Requisites


  • Install eventbrite package pip install --upgrade eventbrite
  • You must have a Eventbrite account.
  • Go to eventbrite apps and create an app
  • After app creation copy and save your personal auth token as you can see in image


Sync Eventbrite:


  • Go to Eventbrite app in ODOO and click SYNC EVENTBRITE button
  • This will import all event, event tickets, attendees, orders from Eventbrite to ODOO
  • Click this button again if you want to sync data again


Eventbrite Tickets


  • Go to Event app in ODOO and open an event synced from eventbrite
  • You can see tickets off event synced as well in Tickets tab


Eventbrite Orders


  • After you performed sync. You can see sale orders of tickets sold in ODOO sales app.
