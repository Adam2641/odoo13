Alfresco Odoo Connector

Document Management System using Alfresco Odoo Connector

Overview

Odoo ERP, is the most popular open source customer relationship management tool and Alfresco is one of the most popular document management and enterprise content management tool. Odoo integration with Alfresco, is a leading giant of Enterprise Content Management (ECM) and Business Process Management (BPM) system, to enhance operation of organization with legitimate documentation. This is done by using Content Management Interoperability Services (CMIS).

Key Functions

  • Upload documents to Alfresco using Odoo
  • Download documents to Odoo from Alfresco
  • Manage separte account for each user

Prerequisites

Windows

  • Install python if not installed
  • Install setuptools and cmislib and iso8601 libraries using this command(pip3 install git+https://github.com/apache/chemistry-cmislib.git@py3_compat#egg=cmislib)

Linux

  • Install setuptools and cmislib and iso8601 libraries using this command(pip3 install git+https://github.com/apache/chemistry-cmislib.git@py3_compat#egg=cmislib)

Alfresco Settings

  • Alfresco credential management for each user
  • Alfresco test connection option

ODOO File Upload

  • Add attachments
  • Separate directory structure for each customer provides an easy way to locate and access files.

ODOO File Download

Sync document between ODOO and alfresco using download document option.

Odoo Files upload in Specific Folder

To Upload Documents into specific Directory Go to Alfresco Documents

  • Select Directory Name
  • Add attachments
  • Click On Upload Button.

Alfresco Repositroy

A directory is created with Odoo client Document and inside this directory the sub directory is created with customer name. All documents are placed inside customer directory.

All other documents are placed inside Odoo Document directory.
