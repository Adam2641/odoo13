Box ODOO Connector

Document Management using Box ODOO Connector

Overview

ODOO is a fully integrated suite of business modules that encompass the traditional ERP functionality. Box is a document management and storage system. Box is a web-based application that integrates with Microsoft Office and it is highly configurable and usage varies substantially between organizations. ODOO integration with Box enhances operation of organization with legitimate documentation.

Key Features And Functions

  • Secure document management
  • Upload ODOO's document to Box
  • Download Box's document to ODOO
  • Directory structure management for multiple customers using ODOO
  • Manage seperate Box account for each user
  • Support for both Enterprise and Community ODOO version
  • Support for multiple Box sites
  • Support for custom documents library

Box Credentials

  • Box credentials management for each user.
  • Box credentials verification using test Connection.
  • Box subsite feature
    • User can upload/download files to his own created subsite
    • If no subsite is mentioned then default site will be used
  • Box document library feature
    • User can upload/download files to their custom libraries
    • If no document library is mentioned then default "Documents" Library will be used
    • User having Box account in a language other than English can use this feature. They just need to mention document library name in their language.

ODOO File Upload/Download

  • Integration of ODOO document with Box for each customers
  • Multiple attachments
  • Download latest files from Box to ODOO for each customer.
  • Download related files links from Box to ODOO for each customer, and each customer can download file from the Download URL when required.

ODOO Documents links

  • Download related files links from Box to ODOO for each customer.
  • Each customer can download file from the Download URL when required.

Documents on Box

  • Directory structure for each customer at Box.
  • Management of sites and document libraries on Box.
