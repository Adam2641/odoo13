Dropbox ODOO Connector

ODOO is a fully integrated suite of business modules that encompass the traditional ERP functionality. Dropbox is a document management and storage system. Dropbox is a web-based application that integrates with Microsoft Office and it is highly configurable and usage varies substantially between organizations. ODOO integration with Dropbox enhances operation of organization with legitimate documentation.



Demo Video





Key Features And Functions

  • Secure document management.
  • Upload ODOO's document to Dropbox
  • Download Dropbox's document to ODOO
  • Directory structure management for multiple customers using ODOO
  • Manage seperate Dropbox account for each user
  • Support for both Enterprise and Community ODOO version
  • Support for multiple Dropbox sites
  • Support for custom documents library
  • Dropbox credentials management for each user.
  • Dropbox credentials verification using test Connection.
  • Dropbox document library feature
    • User can upload/download files to their custom libraries
    • If no document library is mentioned then default "Documents" Library will be used
    • User having Dropbox account in a language other than English can use this feature. They just need to mention document library name in their language.


Dropbox Credentials

  • Dropbox credentials management for each user.
  • Go to DropBox app from ODOO HOME
  • Create an account settings
  • Give name to this dropbox account
  • Give Dropbox folder name. If you want to use subdirectories give folder name like this (main_directory/child_directory)
  • Now you have to provide access token. Got to APP CONSOLEand create a new app. Inside app click "Generate Access Token" button. Copy this token and paste it on ODOO and click TEST CONNECTION button.


ODOO File Upload/Download




  • Integration of ODOO document with Dropbox for each customers
  • Multiple attachments
  • Download latest files from Dropbox to ODOO for each customer
  • Download related files links from Dropbox to ODOO for each customer, and each customer can download file from the Download Document button when required.


ODOO Documents links




  • Download related files links from Dropbox to ODOO for each customer.
  • Each customer can download file from the Download Document button when required.


Documents on Dropbox


  • Directory structure for each customer at Dropbox.
  • Management of sites and document libraries on Dropbox.


Setting Rights for other Users to use Dropbox functionality


  • Enable Developer Mode.
  • Go to Settings --> groups --> Open any Group for which need to add Rights.

  • Add Dropbox credential to Access Rights in selected Group.
  • Directory structure for each customer at Dropbox.
  • Management of sites and document libraries on Dropbox.
