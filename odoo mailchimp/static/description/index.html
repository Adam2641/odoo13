Odoo MailChimp

Odoo is a fully integrated suite of business modules that encompass the traditional ERP functionality. Use Odoo MailChimp to import / export lists between MailChimp and odoo and send Campaigns. Odoo MailChimp provides functionality to import/export lists and send campaigns.



Demo Video





Key Features And Functions

  • Import/Update lists from MailChimp to ODOO.
  • Update lists from ODOO to MailChimp.
  • Add/Remove Customers to a list.
  • Import Campaigns from MailChimp to ODOO.
  • Replicate Campaign.


Entering MailChimp Credentials In Your odoo user account

  • You can enter MailChimp credentials in two places.
  • Go to MailChimp-> MailChimp settings or go to the Settings -> Users. Then provide MailChimp credentials against ODOO's current user.
    • username(username of your MailChimp account)
    • api_key (In MailChimp go to profile --> account --> Extras --> API Keys)
  • Also test these credentials by click on Test Connection button. Save Record.


Go to MailChimp module. You can see the view where you can perform following actions




  • Import campaigns from MailChimp to ODOO.
  • Replicate campaign.
  • Import/Update MailChimp lists.


Mailchimp Campaigns Details




  • Go to MailChimp Campaign Tab where you can see each campaign details.


Create/Update MailChimp list




  • Go to MailChimp lists and create a list. After entering data click SYNC WITH MAILCHIMP button placed in the header.


Customer Details




  • Go to Sales > order >customers where you can see details of Campaigns and lists of each customer.
