Shopify Odoo Connector

Shopify is one platform with all the ecommerce and point of sale features you need to start, run, and grow your business. Odoo is a fully integrated suite of business modules that encompass the traditional ERP functionality. Odoo Shopify provides functionality to import orders and customers from your Shopify shop. It also provides functionality to export products from odoo to your Shopify shop.



Demo Video





Key Features And Functions

  • Export and update products and product details.
  • Export and update product stock, price and images.
  • Import and update customer details.
  • Import and update sale orders.
  • All export and import functionality can be done manually.
  • All export and import functionality can be done automatically.
  • Set automatic job for export and import functionality.
  • Set automatic job(s) individually or collectively for export and import.


Entering Shopify Credentials In Your odoo user account

  • You can enter shopify credentials in two places.
    • Go to Shopify-> Shopify settings.
    • Go to Settings -> Users.
  • In Odoo, map the following things from Shopify.
    • Shop Name
    • Secret Key
    • API key
    • Password
  • Also test these credentials by clicking Test Connection button


Sync Data




  • Manual Export & Update Products From ODOO to Shopify. Note: you have to enable "Sync to Shopify" checkbox in products you want to export
  • Manual Import & Update Sale Orders from Shopify to ODOO.
  • Manual Import & Update Customers from Shopify to ODOO.
  • You can see sync history logs.


Automated Actions Settings




ODOO Shopify Connector also provides automatic jobs to export stock and import sale orders and customers. You can activate a job by clicking respective checkbox. You can also set time after which job will start automatically. After activating a job you can see auto sync logs in Connector History Details section.
